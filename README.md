
**Project Name:** Meta Gen
**Version:** 1.0.0
**Tech Stack:** Vanilla JavaScript (ES6+), HTML5, CSS3

## Executive Summary
**Meta Gen** is a client-side, dependency-free utility designed to generate semantically correct and secure HTML meta tag snippets. It solves the problem of manual syntax errors and repetitive boilerplate writing by providing a clean interface to input SEO, Open Graph, and Twitter Card data, outputting a sanitized, ready-to-paste code block or a downloadable HTML snippet.

## Target Audience
**Meta Gen** is designed for non-technicial users such as marketing personnel and new developers.
